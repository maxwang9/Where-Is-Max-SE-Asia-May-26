# Deploy to GitHub Pages via Web Browser (No Command Line!)

This is the easiest way to deploy your travel tracker - just use your web browser!

---

## Step 1: Create a GitHub Repository

1. Go to [github.com](https://github.com) and **sign in**
2. Click the **"+"** icon in the top-right corner
3. Click **"New repository"**
4. Fill in:
   - **Repository name**: `southeast-asia-travel-tracker`
   - **Description**: "Interactive travel tracker for Singapore & Thailand 2026"
   - **Public** (so you can use free GitHub Pages)
   - ✅ **CHECK** "Add a README file"
5. Click **"Create repository"**

---

## Step 2: Upload Your Files via Web

1. In your new repository, click **"Add file"** button
2. Select **"Upload files"**
3. Drag and drop these files from your computer:
   - `index.html`
   - `README.md`
   - `GITHUB_PAGES_SETUP.md` (optional)
   
   **Location on your computer:**
   ```
   C:\Users\maxjwang\Documents\Projects\2026 Southeast Asia Travel Tracker\
   ```

4. In the commit message box at the bottom, type:
   ```
   Add travel tracker website with mobile optimizations
   ```

5. Click **"Commit changes"**

✅ Your files are now on GitHub!

---

## Step 3: Enable GitHub Pages

1. In your repository, click **"Settings"** tab (top of page)
2. Scroll down and click **"Pages"** in the left sidebar
3. Under **"Branch"**, select:
   - Branch: **`main`**
   - Folder: **`/ (root)`**
4. Click **"Save"**

A green box will appear with your website URL!

---

## Step 4: Access Your Live Website

After 1-2 minutes, your website will be live at:

```
https://YOUR-USERNAME.github.io/southeast-asia-travel-tracker/
```

**Example:**
```
https://maxjwang.github.io/southeast-asia-travel-tracker/
```

Click the link in the green box on the Pages settings page to visit your site!

---

## Making Updates (Web Browser Method)

### To Edit a File:

1. Go to your repository on GitHub
2. Click on the file you want to edit (e.g., `index.html`)
3. Click the **pencil icon** (✏️) in the top right that says "Edit this file"
4. Make your changes
5. Scroll to bottom, add a commit message like "Updated flight times"
6. Click **"Commit changes"**

Your site will automatically update in 1-2 minutes!

### To Upload New Files:

1. Click **"Add file"** → **"Upload files"**
2. Drag and drop the new files
3. Add commit message
4. Click **"Commit changes"**

## Troubleshooting

### Site shows 404 error
- Wait 2-3 minutes after enabling Pages
- Make sure your file is named `index.html` (not `travel-tracker.html`)
- Check that Pages is enabled in Settings → Pages

### Changes not showing
- Wait 2 minutes for GitHub to rebuild
- Hard refresh: `Ctrl + Shift + R`
- Check "Actions" tab for deployment status

### Can't find Settings tab
- Make sure you're signed in
- Make sure you're in YOUR repository (not someone else's)
- Click on the repository name to get to the main page first
